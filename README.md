# Sol- och batteriekonomi

Version: 0.3.3 - Beta med STORA bokstäver

## Enkel uppföljning av den ekonomiska nyttan för en solcellsinstallation med batteri.

Integrationen är just nu totalt avhängig att du har följande mätvärde i din Home Assisstant installation

Mätare för:

- Inköpt energi
- Såld energi
- Egenkonsumerad energi
- Producerad energi
- Energi lagrad i batteri
- Energi använd från batteri
- Spotprissensor

Vidare är det bra om man vet exakt hur prisberäkningen för nätnytta samt överföringsavgift fungerar hos det egna nätbolaget.

Om alla konfigurationsval fylls i kommer det skapas sensorer för
 - Aktuell ersättning för såld kWh
 - Aktuell nätnytta
 - Aktuell Överföringsavgift
 - Aktuellt inköpspris
 - Datakvalite
 - Ekonomisk nytta idag / innevarande månad / totalt
 - Exportintäkt idag / innevarande månad / totalt
 - Importkostnad idag / innevarande månad / totalt
 - Nätnytta idag / innevarande månad / totalt
 - Värde egenförbrukning idag / innevarande månad / totalt
 - Engångsavräkning
 - Kvarvarande investering
 - ROI %
 - Verkningsgrad på batteri

Jag är medveten om att det finns flera brister, framför allt funkar det inte utan batteri och just nu är det moms inkluderat på exproterad el.
Stay tuned, det kommer förbättringar.
